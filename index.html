API Usage:

  1. POST /asm : upload assembly code, via raw post body or form-data file (key="file")

  2. POST /elf : upload executable elf file via form-data file (key="file")

  3. POST /input : give stdin via raw post body or form-data file (key="file"), and then execute elf

  4. GET /output : get output of last run

  5. GET /perf : get perf(stderr) of last run
