Hello, {hostname}

API Usage:

  1. POST /asm : upload assembly code

  2. POST /elf : upload executable elf file

  3. POST /input : send stdin, and then execute elf

  4. GET /output : get output(stdout + return code) of last run

  5. GET /perf : get perf(stderr) of last run

The format of POST body should be raw text or binary, do not use form-data.
